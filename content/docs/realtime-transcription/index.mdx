Demo of MeetingBaas's capablitites to set up real-time meeting transcription, using a third party API (Gladia) to transcribe the audio.
This demo logs the transcriptions to the terminal and to a file.

## How It Works

```
Meeting ─────┐
            Audio Stream
                │
                ▼
        ┌─── MeetingBaas ───┐
        │                   │
     Audio              Transcript
        │                   │
        ▼                   ▲
    Gladia API ────────────┘
        │
        ▼
     Storage
```

## Key Features

- **Universal Meeting Support**: Works with Zoom, Google Meet, Microsoft Teams
- **Real-time Processing**: Live audio streaming and transcription
- **Speaker Detection**: Identifies and labels different speakers
- **Easy Integration**: Simple setup and configuration
- **Flexible Deployment**: Run locally or in the cloud

## Use Cases

- Meeting documentation and notes
- Accessibility and transcription
- Knowledge management
- Compliance and auditing
- Team collaboration

## Quick Start

```bash
# Install
git clone https://github.com/Meeting-Baas/realtime-meeting-transcription.git
cd realtime-meeting-transcription
pnpm install

# Configure
cp .env.example .env
# Edit .env with your API keys

# Run
pnpm start
```
