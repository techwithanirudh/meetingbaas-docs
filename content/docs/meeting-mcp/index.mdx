The Meeting MCP (Model Context Protocol) server is a powerful tool that enables AI assistants to seamlessly interact with video meetings. It provides a standardized interface for creating meeting bots, managing transcripts, and handling meeting recordings across multiple platforms.

## Key Features

- **Universal Meeting Support**: Works with Zoom, Google Meet, Microsoft Teams
- **Real-time Processing**: Live audio streaming and transcription
- **Speaker Detection**: Identifies and labels different speakers
- **Calendar Integration**: Schedule and manage recordings
- **Search & Analytics**: Find and analyze meeting content

## Use Cases

- Meeting documentation and notes
- Accessibility and transcription
- Knowledge management
- Compliance and auditing
- Team collaboration

## Quick Start

```bash
# Install
git clone https://github.com/Meeting-Baas/meeting-mcp.git
cd meeting-mcp
pnpm install
pnpm build

# Run
pnpm start
```
