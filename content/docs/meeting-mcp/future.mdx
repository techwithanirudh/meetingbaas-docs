## SDK Generator (WIP)

[View on GitHub](https://github.com/Meeting-Baas/sdk-generator)

```bash
# Prerequisites
node >= 18
pnpm >= 8
ANTHROPIC_API_KEY=your_key_here

# Build process
pnpm openapi:clean    # Clean generated files
pnpm openapi:generate # Generate TypeScript client
pnpm build           # Build SDK
pnpm tools:generate  # Generate MCP tools
pnpm tools:register  # Register with MCP server
```

### Features

- OpenAPI to MCP tool generation
- TypeScript types and validation
- Custom tool templates
- Claude-powered definitions
- Local/hosted deployment options
